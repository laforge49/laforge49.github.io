Project JActor2: Documentation & Downloads

A Robust Alternative to Threads and Locks

JActor2 is a multi-threaded OO programming model, inspired by Alan Kay's early thoughts on [Objects](http://c2.com/cgi/wiki?AlanKaysDefinitionOfObjectOriented). JActor2 is based on asynchronous 2-way messaging with assured responses. The net result being code that is both simpler and more robust, and hence easier to maintain.
